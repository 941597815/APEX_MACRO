Apex-Mcaro
功能：抖枪 换甲 拾取 变向 SG
需要硬件 Arduino Pro Micro / RP2040

#### 软件安装

安装 python >= 3.10.11

安装项目依赖 requirements.txt 若缺少模块请手动安装

```bash
pip install -r requirements.txt -i https://mirrors.huaweicloud.com/repository/pypi/simple
```


#### 运行

连接硬件后运行main.py

###### HOME/断开硬件 退出程序

###### 配置文件 config.yaml
